Soft, rounded content surface — the default container for editorial blocks. Low warm shadow or hairline border, large radius.

```jsx
<Card variant="raised" radius="lg">…</Card>
<Card variant="deep" radius="xl" interactive>…</Card>
```

Variants: `raised` (paper + soft shadow), `flat` (hairline border), `sunken`, `accent` (turquoise), `deep` (dark turquoise), `inverse` (black). Set `interactive` for hover lift on clickable cards. `radius` `md | lg | xl | 2xl`.
