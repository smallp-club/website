Primary action element — an obround (pill) button derived from the brand mark; calm motion, one accent per screen.

```jsx
<Button variant="primary" size="md">read the facts</Button>
<Button variant="secondary">learn more</Button>
<Button variant="ghost" size="sm">skip</Button>
```

Variants: `primary` (turquoise), `secondary` (outlined black), `ghost` (text-only), `deep` (dark turquoise), `signal` (burnt sienna — rare emphasis only). Sizes `sm | md | lg`. Use `shape="square"` for a soft-radius rectangle; default `shape="pill"`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`.
