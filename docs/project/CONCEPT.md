# Konzept — Landing Page

## Status: Phase 1 abgeschlossen (2026-06-12)

## Story Architecture

### Emotionale Reise (5 Kapitel)

```
1. ANKUNFT
   "Wir reden über das hier."
   Kein Aufwärmen. Kein Erklären. Direkt.
   Herb, ruhig, selbstbewusst. Der Besucher weiß sofort: kein Wellness-Portal.

2. ERKENNTNISMOMENT
   "Das ist größer als du dachtest."
   Von der privaten Männerrunde zur gesellschaftlichen Realität.
   Fakten — nicht als Schock, als stille Klarheit.

3. MYTHOS BRICHT (Signature Moment)
   "Das stimmt nicht. Das haben wir uns eingebildet."
   Mythos steht groß im Raum → User scrollt → Wahrheit ersetzt ihn still.
   Wiederholt sich 3–4x mit verschiedenen Mythen/Fakten.

4. DU BIST NICHT ALLEIN
   Betroffene UND Anhänger. Beide willkommen.
   Community-Signal — man muss nicht betroffen sein um dazuzugehören.

5. MITMACHEN
   Newsletter = Membership. Kostenlos. Kein Gate.
   Einfach: dabei sein.
```

### Signature Moment — Mythos-Reveal (Detail)
- Mythos steht typographisch groß auf dem Screen
- Beim Scrollen beginnt er sich still zu verändern (kein Drama, keine Explosion)
- Fakt bleibt stehen — herb, knapp, mit Quellenangabe
- Beispiel:
  - Mythos: *"Ein kleiner Penis macht einen Mann weniger attraktiv."*
  - Fakt: *"67% der Partner nennen Größe nicht unter den Top-5-Faktoren. Studien. Plural."*
